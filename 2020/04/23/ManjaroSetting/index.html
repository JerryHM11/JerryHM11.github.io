Manjaro安装后的初始配置

Mr. X

Manjaro/Linux

发布于：Apr 23, 2020

​ Manjaro是基于Arch Linux的发行版，因此有着Arch所具有的优秀包管理器，并且没有Arch所带来的滚动式更新所带来的不稳定。

​ 最为突出的就是有着非常强大的AUR源，这是个全球用户开发者所共同维护的软件源，为Manjaro的使用不会为软件的匮乏而担心。

​ 由于Manjaro的默认源都是在国外，若是不对其进行相应的配置，作为国内使用者则会使用困难。

1、设置国内源

Ctrl+Alt+T打开终端

修改配置文件，连接国内archlinuxcn源

sudo vim /etc/pacman.conf在文档末尾添加

Esc + :wq 退出保存
